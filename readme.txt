== Description ==
Minimal valid skeleton plugin to use as a template/wrapper for Superdev output.

== Installation ==
1. Upload the `vibe-coding-seo-plugin.zip` via Plugins > Add New > Upload Plugin.
2. Activate the plugin from the Plugins screen.
